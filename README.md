## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn
- MongoDB instance
- Clerk account
- Cloudinary account

### Installation Steps
1. Clone the repository
   ```bash
   git clone https://github.com/vipulkatwal/carvault.git
   ```

2. Install dependencies
   ```bash
   cd car-manager-pro
   npm install
   ```

3. Set up environment variables
   - `.env.local` for frontend
   - `.env` for backend

   Required variables:
   - `CLERK_PUBLISHABLE_KEY`
   - `CLERK_SECRET_KEY`
   - `MONGODB_URI`
   - `CLOUDINARY_CLOUD_NAME`
   - `CLOUDINARY_API_KEY`
   - `CLOUDINARY_API_SECRET`

4. Run the application
   ```bash
   # Frontend
   npm run dev

   # Backend
   npm run server
   ```
